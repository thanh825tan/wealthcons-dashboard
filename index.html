Wealthcons — Kết nối Google Sheets

Dashboard sẽ đọc dữ liệu trực tiếp từ Google Sheets của bạn. Mọi thay đổi trên Google Sheets sẽ tự động cập nhật trên dashboard.

Bước 1: Import file wealthcons_dashboard_template.xlsx lên Google Sheets
Bước 2: Vào File → Share → Publish to web → chọn Entire Document → nhấn Publish
Bước 3: Copy Spreadsheet ID từ URL:
https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit
Bước 4: Dán ID vào ô bên dưới và nhấn Kết nối
Không thể kết nối. Kiểm tra lại ID và đảm bảo đã Publish to web.
